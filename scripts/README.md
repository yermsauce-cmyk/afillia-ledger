# Scripts

Helper scripts for the Afillia TikTok policy + compliance agent.

- `compliance_check.py` — pre-post compliance checkpoint. Run before publishing any caption.
- `scrape_tiktok_guidelines.py` — fetches TikTok guidelines + Reddit/X chatter into `logs/`.

Add more helpers here as the workflow grows (e.g. Slack/email alerter, n8n webhook receiver).
